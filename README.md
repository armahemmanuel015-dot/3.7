# Block 3.7 Deep Tutor

Single-file interactive HTML revision app for Prof. Edmund Muonir Der's Block 3.7 MCQs.

- Prof. Der's Block 3.7 slides are the primary answer authority.
- Pathology Secrets, 3rd ed. is used as fallback/deepening source.
- The Masterpiece MCQ PDF is used for question stems only; its supplied answers are ignored.

Open `index.html` directly or deploy the repository with GitHub Pages.
